Пространства
В распоряжение СК Олимп находятся 5 открытых спортивных площадок, 3 крытых многофункциональных зала, 2 тренажёрных зала и профессиональный велотрек
  • Открытые Площадки
    Открытые площадки СК Олимп представляют собой современные пространства для занятий спортом. Каждая из площадок имеет несколько сценариев использования и пригодна для занятий спортом разных направлений таких как: футбол, волейбол, хоккей, баскетбол, воркаут и даже теннис. Площадь пощадок от 20 до 60 квадратных метров. Некоторые из площадок имеют искуственное освещение.
    • Площадка Красная Пахра, 11
      Многофункциональная площадка для занятий игровыми видами спорта. Оборудована футбольными воротами и баскетбольными кольцами
    • Площадка Красная Пахра, 11
      Площадка для занятий силовыми видами спорта и воркаутом. Оснащена современными тренажёрами
    • Площадка Былово, 8А
      Площадка в селе Былово оборудована инвентарём для занятий воркаутом и силовыми тренировками
    • Площадка Красная Пахра, 20
      Площадка для занятий хоккеем в зимний период. Летом покрыта искусственным газоном для занятий футболом
    • Площадка Красная Пахра, 20
      Многофункциональная площадка для занятий спортом. Может использоваться как футбольная и волейбольная площадка или как тенесный корт
  • Крытые Залы
    Крытые залы СК Олимп представляют собой современные пространства для занятий спортом. Каждый зал оборудован для использования в определённых направлениях спорта. Два зала имеют покрытие из матов на полу и предназначены в основным для боевых единоборств. Площадь залов от 30 до 60 квадратных метров. Все залы имеют искуственное освещение.
    • Зал Красная Пахра, 11
      Зал для занятий боевыми единоборствами. Покрытие пола - маты. Может использоваться как зал для кардио тренировок.
    • Зал Красная Пахра, 20
      Зал для занятий волейболом. Покрытие пола - паркет. Оборудован волейбольной сеткой. Также используется для занятий футболом.
    • Зал Былово, 8А
      Зал для занятий боевыми единоборствами. Покрытие пола - маты. Может использоваться как зал для кардио тренировок.
  • Тренажёрные залы
    Тренажёрные залы СК Олимп представляют собой современные пространства для занятий силовыми видами спорта. Каждый из залов оборудован специализированнми тренажёрами и большим ассортиментом силового инвентаря. Площадь залов 25 квадратных метров. Все залы имеют искуственное освещение.
    • Тренажёраный зал Красная Пахра, 11
      Тренажёрный зал оснащён всем необходимым инвентарём для занятий силовыми тренировками. Имеются тренажёры для кардио нагрузок.
    • Тренажёрный зал Красная Пахра, 20
      Тренажёрный зал оснащён всем необходимым инвентарём для занятий силовыми тренировками. Имеются тренажёры для кардио нагрузок.
  • Велотрек
    Велотрек СК Олимп представляет собой современную площадку для занятий велогонками BMX. Один из лучших треков Москвы и ТиНАО. Неоднократно принимал крупные Московские соревнования по велогонкам. Трек грунтовый с искусвенно возведёнными холмами. За поверхностью велотрека ведётся постоянный уход. Трек оборудован трибунами для болельщиков. Имеет искуственное освещение.
