# ATH Store — Apple Tech Shop

Сайт интернет-магазина Apple-техники, задеплоенный на GitHub Pages.

## Стек

- **React 19** + **TypeScript**
- **Vite** — сборка
- **TanStack Router** — роутинг
- **Tailwind CSS v4** — стили
- **Supabase** — бэкенд
- **Bun** — пакетный менеджер

## Локальный запуск

```bash
bun install
bun run dev
```

Сайт откроется на `http://localhost:5173`.
